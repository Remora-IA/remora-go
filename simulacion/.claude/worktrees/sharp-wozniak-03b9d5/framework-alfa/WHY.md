# WHY - Framework Alfa

Alfa existe porque un árbol de descubrimiento no es una especificación
ejecutable.

Echo descubre dolores y oportunidades. Alfa traduce eso en un flujo ideal
con reglas verificables, variables trazables y un modelo de datos normalizado
que Bravo pueda comparar contra un trace real.

## Problema Que Resuelve

Sin Alfa, pasar de "entendimos el proceso" a "tenemos un spec para codificar"
requiere un humano que interprete notas sueltas y las convierta en reglas.
Alfa hace esa compilación de forma estructurada y trazable.

## Relación Con Otros Frameworks

- **Echo** provee el árbol validado que Alfa consume.
- **Bravo** verifica que el código real siga el flujo ideal que Alfa genera.
- **Foco** puede pedir a Alfa un draft temprano cuando hay TASK + PAIN.

Alfa no descubre. Echo no compila. Bravo no genera flujos.
