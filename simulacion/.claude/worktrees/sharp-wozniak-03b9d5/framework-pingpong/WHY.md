# WHY: framework-pingpong

## Proposito

Framework framework-pingpong creado para tutor iterativo de programación

## Tipo

inquisitivo

## Rol operativo

tutor iterativo de programación

## Limite de razonamiento

La IA operadora no debe inventar pasos internos. Debe ejecutar los comandos del framework y dejar que el codigo aplique las reglas de negocio.

## Criterio de funcionamiento

El framework funciona si `go test ./...` pasa y sus comandos principales responden sin depender de instrucciones narrativas del prompt.
