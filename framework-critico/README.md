# Framework Critico

Framework adversarial para evaluar propuestas de cambio.

## Comandos

| Comando | Proposito |
|---------|-----------|
| `init` | Crea sesion |
| `evaluate` | Evalua propuesta contra modelo del repo |
| `challenge` | Cuestiona una asuncion con evidencia |
| `status` | Estado de la sesion |
| `readiness` | Accion recomendada |
| `next-question` | Proxima pregunta pendiente |
| `ingest-answer` | Registrar respuesta |

## Estado

El estado se guarda en `temp/critico_state.json`.

## Principio

No se trata de ser negativo. Se trata de que ninguna asuncion pase sin ser cuestionada.
